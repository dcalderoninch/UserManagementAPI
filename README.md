## Description 📖
API for TechHive Solutions whose functionality is basic user management.

## Project Setup ⚙️
Git:

    git clone https://github.com/dcalderoninch/UserManagementAPI.git
    cd UserManagementAPI

or download the [zip file](https://github.com/dcalderoninch/UserManagementAPI/archive/refs/heads/master.zip).

## Utilities 🔍
The repository includes the file **requests.http**, which is intended to help test API consumption.

## Project Execution ▶️
1. Compile the project:

```
dotnet build
```

2. Run the project:
```
    dotnet run
```
3. Use the **requests.http** file to test the API endpoints.
